  • AoE2Tools

    Aoe2Tools is an awesome piece of software by Gregstein that makes it as easy as never before to install and manage your Age of Empires II game, including WololoKingdoms, Visual Mods, Recorded Games, Streams and much more!

  • WololoKingdoms

    This tool allows you to play the HD Edition DLCs on Voobly and with Userpatch, so you can profit from a huge variety of features, bug fixes and improvements. Even the ECL (Escape Champions League), a 60000$ tournament official sponsored by Microsoft, is played on this version.

  • OBS Studio AoC edition

    This is a custom version of OBS, which is streaming software. This version has useful scene presets with a score overlay and more. It also has custom encoder settings which should improve the stream quality for AoE II.

  • AoE 2 Stats

    This is a great site made by Pip. Similar to my Sheet, you can look at unit stats, civilizations and more. The search function makes it very convenient and there's a option to compare several units that's very useful!

  • AoE 2 Map Database

    A database of different maps and map packs, with descriptions, screenshots a map pack creator and more!

    Back to top

  • Online Hotkey Editor

    An online tool for editing hotkeys. There are no restrictions on overlapping keys, and you can set some unusual keys that the in-game editor won't allow. You can find presets from pro players as well. When downloading, make sure to set the version to HD 5.0+ if you want to use the hotkey set for HD or WololoKingdoms.

    Back to top

  • Captains Mode for Age of Empires II

    Another great tool by Pip. This allows you to pick and ban civs according to certain rules, as you might know from games like League of Legends. This has been used in several tournaments already.

  • AoE 2 Wiki

    A wiki site for the Age of Empires series. Keep in mind that as a wiki it's editable by anyone and relies on different volunteers for updating.
